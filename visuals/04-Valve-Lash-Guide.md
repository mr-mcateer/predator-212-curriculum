## ğŸ“ Finding Top Dead Center (TDC)

This is where both valves are fully closed and the piston is at the top.

### **Step 1: Locate Flywheel**
The flywheel is the big wheel on the crankshaft end (where you attached the pull cord).

### **Step 2: Mark on Flywheel**
```
Some engines have marks, some don't.
Look for:
  - "TDC" label
  - A line marked in paint
  - A notch

If no mark exists:
  - You'll use the pushrods as indicator
  - When both are loose, you're at TDC
```

### **Step 3: Find TDC Position**

**Method A (If you have marks):**
- Rotate flywheel until mark aligns with fixed point on engine
- Both valves should be closed
- Verify by trying to move pushrods - they should have slight play

**Method B (Using pushrods):**
```
1. Look at pushrods on top of engine
2. Slowly rotate engine by hand
3. One pushrod will start lifting up
4. Continue rotating until it peaks and starts going down
5. At the peak, continue rotating slowly
6. Continue until BOTH pushrods are completely down (loose)
7. That position = TDC
```

## ğŸ“ Troubleshooting

### **Problem: Feeler gauge won't fit in gap**

```
Diagnosis: Valve lash too tight

Solution:
1. Loosen rocker arm bolt
2. Slide rocker ARM AWAY from valve
3. This increases the gap
4. Tighten bolt and re-check
5. Try gauge again
```

### **Problem: Feeler gauge slides in and out too freely**

```
Diagnosis: Valve lash too loose

Solution:
1. Loosen rocker arm bolt
2. Slide rocker arm TOWARD valve
3. This decreases the gap
4. Tighten bolt and re-check
5. Try gauge again
```

### **Problem: Gauge fits sometimes but not always**

```
Diagnosis: Inconsistent adjustment

Solution:
1. Engine might not be at TDC
2. Rocker bolt might be loose
3. Rocker arm isn't positioned correctly

Fix:
1. Re-verify you're at TDC
2. Tighten all rocker bolts firmly
3. Start over with adjustment
```

### **Problem: Values change when you rotate engine**

```
Diagnosis: Rocker bolt came loose

Solution:
1. Go back and tighten the rocker bolt
2. Re-verify lash with gauge
3. Re-check after tightening
```
